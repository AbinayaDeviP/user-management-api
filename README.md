# User Management API

## Setup
1. Install Node.js and MongoDB
2. Run: npm install
3. Update .env file
4. Run: npm start

## APIs
POST /api/users
GET /api/users?page=1&limit=10&search=name
GET /api/users/:id
PUT /api/users/:id
DELETE /api/users/:id

## Features
- Pagination
- Search
- Soft delete
- Rate limiting
